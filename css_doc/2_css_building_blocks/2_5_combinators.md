# 결합자

<br/>

> [목표] 결합자는 다른 선택자를 서로 유용한 관계와 문서의 content 위치를 제공하는 방식으로 결합하기 때문입니다.

<br/>

## 후손 결합자

<b>일반적으로 단일 공백("") 문자로 표시됩니다</b> - 첫 번째 선택자와 일치하는 부모 요소가 있는 경우 두 번째 선택자와 일치하는 요소가 선택되도록 두 선택자를 결합합니다. 후손 결합자를 활용하는 선택자를 <b style="color: orange;">하위 선택자</b>라고 합니다.

```css
body article p
```

<br/>

## 자식 결합자

<b>자식 결합자(>)는 두 CSS 선택자 사이에 배치됩니다.</b> 첫 번째와 일치하는 직계 자식의 두 번째 선택자와 일치하는 요소만 일치합니다. 계층 구조에서 더 아래에 있는 하위의 요소는 일치하지 않습니다.

```css
article > p
```

<br/>

## 인접 형제 결합자

<b>인접한 형제 선택자(+)는 두 CSS 선택자 사이에 배치됩니다. 첫 번째 선택자의 다음 형제 요소인 두 번째 선택자와 일치하는 요소만 일치합니다.</b>

```css
p + img
```

<br/>

## 일반 형제 결합자

<b>요소의 형제 요소가 바로 인접하지 않더라도 선택하려면 일반 형제 연결자(~)를 사용할 수 있습니다.</b> `<p>` 요소 다음에 오는 모든 `<img>` 요소를 선택하려면 다음을 수행합니다.

```css
p ~ img
```

<br/>

## 결합자 사용

문서의 일부를 선택하기 위해 선택자를 연결자와 결합할 수 있습니다.

```css
/* <ul>의 직계 자식인 “a” class가 있는 목록 항목을 선택 */
ul > li[class="a"] {
}
```

그러나, 문서의 특정 부분을 선택하려면 큰 선택자 목록을 만들 때는 주의하세요. markup에서 해당 요소의 위치에 매우 특정한 선택자를 만들었기 때문에 CSS 규칙을 재사용하기 어려울 것입니다.

간단한 class를 만들고 해당 요소에 적용하는 것이 더 나은 경우가 있습니다. 즉, 결합자에 대한 지식은 문서에서 스타일을 지정해야 하고 HTML이 CMS에 의해 생성되기 때문에 HTML에 Access할 수 없는 경우 매우 유용할 것입니다.
